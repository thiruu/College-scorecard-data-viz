Improved data visualization for College Scorecard results page (R programming & Tableau)

? Analyzed 2013 college scorecard data-set from US Dept. of education using R and graphically represented variables using tableau.

? Using R, cleansed the data and ran descriptive statistics to understand the spread and the characteristics of the data-set. The results page of college scorecard site shows basic info in the form of bar charts about colleges that satisfy the user’s search criteria.

? Using tableau, a map based data visualization was created that would let the users to interactively compare colleges in a more visually appealing manner thereby helping the students and parents in taking quicker informed decisions.
